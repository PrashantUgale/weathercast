# weathercast

The purpose for creating this demo to impliment the functionality for showing weather details for multiple cities.


Functionalities :-

    1. Get input for multiple cities from the user.
    2. Creating URL connection.
    3. Send request for weather details and collect responce.
    4. Parse JSON responce and show 14 days weather details in UITableView.
    5. Also show the detailed information for each day like Tempareture status, Pressure, amount of clouds, speed of wind, etc.
